# Play Json

From https://www.playframework.com/documentation/2.6.x/ScalaJson

## Domain

Play json provides 3 types:
- JsValue: a trait representing any Json value.
    - JsString
    - JsNumber
    - JsBoolean
    - JsArray
    - JsObject
    - JsNull
- Json: utility object for json conversion
- JsPath: a path into JsValue
